# Data Analytics Portfolio

This repository contains multiple data analytics and data science projects demonstrating skills in:
- Exploratory Data Analysis (EDA)
- Customer retention analysis
- RFM customer segmentation
- Business insights and recommendations
- Python (Pandas, NumPy, Matplotlib, Seaborn)

## Projects Included

### 1. Retail Sales Exploratory Data Analysis
- Cleaned and analyzed transactional retail data
- Identified revenue drivers, top products, country-wise performance, and seasonality

### 2. Customer Retention Analysis
- Built customer-level metrics from transactional data
- Analyzed Recency, Frequency, Total Spend, and Repeat Customers
- Segmented customers into value tiers

### 3. Customer Segmentation using RFM
- Applied RFM (Recency, Frequency, Monetary) analysis
- Classified customers into actionable segments:
  - Loyal Customers
  - Potential Loyalists
  - At Risk
  - Low Value
